    background-colors

    @mixin background-colors($list: $background-color-names, $is-appended: $modifier-is-appended, $delimiter: $modifier-delimiter) { ... }

    Description

    Apply all colors as modifiers, as background color.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $list

    list of color names

    		List$background-color-names
    $is-appended

    if the selector is appended (or nested)

    		Bool$modifier-is-appended
    $delimiter

    delimiter for class/modifier name

    		String$modifier-delimiter

    Example

    include specific names

    @include background-colors("blue", "black");
// &.blue { color: color(blue); }
// &.black { color: color(black); }

    include all

    @include background-colors;
// &.blue { color: color(blue); }
// &.black { color: color(black); }
// ... /* all the other colors too */

    Output

    modifiers/class names wrapped around background colors

    Requires

    • [mixin] concat
    • [function] color
    • [variable] background-color-names
    • [variable] modifier-is-appended
    • [variable] modifier-delimiter

    get-typography-dimensions-map

    @function get-typography-dimensions-map($name, $breakpoint: null) { ... }

    Description

    Get a full map of a single typography dimension. Optionally get only a specific breakpoint.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $name

    typography dimension name

    		String— none
    $breakpoint

    breakpoint name

    		Stringnull

    Returns

    Map —

    map of typ-dimensions & their values

    Example

    get page-heading

    get-typography-dimensions-map(page-heading);
// (
//   "font-size": 28,
//   "line-height": 36,
//   "letter-spacing": 0.1,
// )

    get page-heading, at medium breakpoint

    get-typography-dimensions-map(page-heading, medium);
// (
//   "font-size": 36,
//   "line-height": 44,
// ),

    Requires

    • [variable] typography-dimensions-map

    Used by

    • [function] get-typography-dimension-value

    typography-modifiers

    @mixin typography-modifiers($list: $typography-dimensions-names, $is-appended, $delimiter) { ... }

    Description

    Apply all typography dimensions to a selector as modifiers, and add all related styles to an element.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $list

    list of typography dimension names

    		List$typography-dimensions-names
    $is-appended

    if the selector is appended (or nested)

    		Bool— none
    $delimiter

    delimiter for class/modifier name

    		String— none

    Example

    include specific names

    @include typography-modifiers("page-heading", "section-heading");
// &--page-heading { /* typography dimensions */ }
// &--section-heading { /* typography dimensions */ }

    include all

    @include typography-modifiers;
// &--page-heading { /* typography dimensions */ }
// &--section-heading { /* typography dimensions */ }
// ... /* all the other modifiers too */

    Output

    modifiers/class names wrapped around styles

    Requires

    • [mixin] concat
